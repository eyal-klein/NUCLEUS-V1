## Architecture

NUCLEUS V1.0 is an **event-driven, multi-agent system** built on a modern Python stack, leveraging:

- **Google ADK (Agent Development Kit)** for agent management
- **LangChain** for tool definitions
- **PostgreSQL + pgvector** for data and semantic search
- **NATS JetStream** for event-driven communication
- **Google Cloud Platform (GCP)** for deployment

For detailed architecture, see:

## Key Features

- **Biological Growth:** The system adapts and evolves as you grow, creating new agents and tools as needed.

- **Full Observability:** Every decision is logged and explainable.
